  • TrustPin

    A namespace for TrustPin core functionality.

    This enum provides APIs to initialize the TrustPin core library and perform certificate verification against domains using public key pinning.

    Use TrustPin.setup(organizationId:projectId:publicKey:) to initialize the library, and TrustPin.verify(domain:certificate:) to perform asynchronous domain and certificate validation.

    Note

    The verification process is asynchronous and utilizes Swift concurrency.
